# My Store Pro Theme

A professional Shopify Online Store 2.0 theme package for testing the VS Code → GitHub → Shopify workflow.

## Includes

- Modern responsive header and footer
- Professional gradient hero section
- Value proposition cards
- Featured products section
- Collection campaign banner
- Testimonials
- Newsletter signup
- Product, collection, cart, page, search and 404 templates

## How to use

### Option 1: Upload to Shopify
Shopify Admin → Online Store → Themes → Import → Upload zip file.

### Option 2: Use with GitHub connected theme
Copy these files into your theme project, then run:

```bash
git status
git add .
git commit -m "[Update]: Add professional full page design"
git push
```

After pushing, Shopify should sync the connected theme from GitHub.
